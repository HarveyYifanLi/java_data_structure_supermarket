This is a realization of a java data structure for a real-world supermarket.

The classes and interface are placed in the package transaction.

The class Person is at the top of the class inheritance hierachy tree;
Class Employee is a subclass of Class Person;
Classes Manager, SalesPerson, and EmployeeCustomer are subclasses of Class Employee;
Besides, EmployeeCustomer implements the interface CustomerInterface;
Class Customer defines the data and behaviour for customers of the supermarket.


The classes Person, Employee, Manager, SalesPerson, and EmployeeCustomer specify data representation
for persons, employees, managers, sales persons, and customers who are employees of the supermarket.

The test class containing the main() method is in file test.java
